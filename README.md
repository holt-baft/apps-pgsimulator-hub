# apps-pgsimulator-hub

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态方式运行，不依赖特定的后端服务或域名环境。仓库的主要目的是提供一种结构化的方式来保存和展示这些 HTML 资源，方便后续查看、测试或部署。

## 目录说明

```
apps-pgsimulator-hub/
├── pages/          # 存放所有 HTML 页面文件
├── assets/         # 页面引用的资源文件（CSS、JavaScript、图片等）
└── README.md       # 本文件
```

- `pages/`：每个 HTML 文件独立存放，文件名通常反映页面功能或内容。
- `assets/`：按需存放页面所需的公共或专用资源。

## 页面归档说明

- 所有页面均为独立 HTML 文件，可直接在浏览器中打开运行。
- 页面之间不共享状态或数据，彼此完全隔离。
- 归档内容可能包含交互界面、模拟工具、演示页面等，具体功能请参考页面文件内的注释或说明。
- 本仓库不保证所有页面在所有浏览器或设备上完全兼容，但尽量使用标准 Web 技术实现。

## 维护说明

- 本仓库以归档为主，不定期更新。
- 新增页面建议放入 `pages/` 目录，并保持文件名简洁。
- 如需引用外部资源，请优先放入 `assets/` 目录，避免依赖外部链接。
- 欢迎提交 Issue 或 Pull Request 来修正问题或改进页面内容。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 在浏览器中直接打开 `pages/` 目录下的任意 HTML 文件。
3. 或将 `pages/` 目录部署到任意静态托管服务（如 GitHub Pages、Netlify 等）进行在线访问。
